Project 1: Defining a Problem and Data Understanding¶

Section 0: Data used in this Project¶

The data set that I will be using is the [Global Music Streaming Listener Preferences Analysis] (https://github.com/SDVNishant9/Global-Music-Streaming-Listener-Preferences-Analysis).

This dataset is a collection of 5,000 users on different music streaming platforms and their preferences for different types of music based on many factors like preferred genres, preferred artists, preferred streaming platforms, country of residence, age, etc. This dataset does contain limitations as this data set was collected from a sample of 5,000 users which is a very small representation compared to the entire population that listens to music. This data is also only collected from streaming services so if a user only listens to music via Radio, they will not be included in this data set which can skew overall results.

Section 0.1: Problem Statement¶

The questions that I primarily hope to answer from this data set is the following:

  • How does a users music preference change as they get older?

  • Do different genres tend to vary in popularity depending on the country of origin?

  • Does time of day help determine the length in which a user will listen to music?

Section 2.1: Genre Preferences as a user gets older.¶

We will start by visualizing the genre preferences as a user gets older using a Heatmap.

# Group the data by age and genre to count the number of users in each genre within each age group
age_genre_counts = music_data.groupby(['Age', 'Top Genre']).size().reset_index(name='User_Count')

# Create a pivot table for the heatmap
heatmap_data = age_genre_counts.pivot(index='Age', columns='Top Genre', values='User_Count').fillna(0)

# Sort genres by total user count across all ages
heatmap_data = heatmap_data[heatmap_data.sum(axis=0).sort_values(ascending=False).index]

# Plot the heatmap
plt.figure(figsize=(12, 8))
sns.heatmap(
    heatmap_data,
    cmap='coolwarm',
    annot=True,
    fmt='.0f',
    cbar_kws={'label': 'Number of Users'}
)

# Titles and labels
plt.title('Music Genre Preferences by Age Group', fontsize=16)
plt.xlabel('Top Genre', fontsize=12)
plt.ylabel('Age Group', fontsize=12)

# Rotate x-axis labels for better readability
plt.xticks(rotation=45)

# Invert y-axis to have older ages at the top
plt.gca().invert_yaxis()

# Show the plot
plt.show()

Section 2.2: Genre Preferences by Country of Origin.¶

We will now visualize the genre preferences by country of origin using a Grouped Bar Plot.

In [7]:
# Group the data by country and genre to count the number of users in each genre per country
country_genre_counts = music_data.groupby(['Country', 'Top Genre']).size().reset_index(name='User_Count')

# Pivot the data to make genres columns for easier plotting
pivot_data = country_genre_counts.pivot(index='Country', columns='Top Genre', values='User_Count').fillna(0)

# Plot the grouped bar plot
ax = pivot_data.plot(kind='bar', figsize=(14, 8), colormap='viridis', width=0.85)

# Calculate the top genre for each country
top_genres = pivot_data.idxmax(axis=1)  # Get the top genre for each country
top_values = pivot_data.max(axis=1)    # Get the corresponding user count for the top genre

# Annotate the top genre on the bar chart
for i, country in enumerate(pivot_data.index):
    ax.text(i, top_values[country] + 1,  # Position the text slightly above the bar
            f'{top_genres[country]}',    # Label with the top genre
            ha='center', va='bottom', fontsize=10, color='black')
    
max_value= int(top_values.max())
plt.yticks(np.arange(20, max_value +5,5))
plt.ylim(20, max_value + 5)
  
# Titles and labels
plt.title('Music Genre Preferences by Country of Origin', fontsize=16)
plt.xlabel('Country', fontsize=12)
plt.ylabel('Number of Users', fontsize=12)

# Customize the legend
plt.legend(title='Top Genre', bbox_to_anchor=(1.05, 1), loc='upper left', fontsize=12, title_fontsize=14)

# Rotate x-axis labels for better readability
plt.xticks(rotation=45)
plt.tight_layout()

# Show the plot
plt.show()

Section 2.3: Genre Preferences by Time of Day.¶

We will now visualize the genre preferences by time of day using a Line Plot.

In [8]:
# Group the data by time of day and genre to count the number of users for each genre at each time
time_genre_counts = music_data.groupby(['Listening Time (Morning/Afternoon/Night)', 'Top Genre']).size().reset_index(name='User_Count')

# Defining the order of the times of day
time_order = ['Morning', 'Afternoon', 'Night']
time_genre_counts['Listening Time (Morning/Afternoon/Night)'] = pd.Categorical(time_genre_counts['Listening Time (Morning/Afternoon/Night)'], categories=time_order, ordered=True)

# Pivot the data to make genres columns for easier plotting
time_genre_pivot = time_genre_counts.pivot(index='Listening Time (Morning/Afternoon/Night)', columns='Top Genre', values='User_Count').fillna(0)

# Use a color palette
colors = plt.cm.tab10.colors  # Use the 'tab10' colormap for distinct colors

# Plot the line plot
plt.figure(figsize=(14, 8))
for i, genre in enumerate(time_genre_pivot.columns):  # Enumerate to get the index for colors
    plt.plot(time_genre_pivot.index, time_genre_pivot[genre], marker='o', label=genre, color=colors[i % len(colors)], linewidth=3)

# Titles and labels
plt.title('Music Genre Preferences by Time of Day', fontsize=16)
plt.xlabel('Time of Day', fontsize=12)
plt.ylabel('Number of Users', fontsize=12)

# Customize the legend
plt.legend(title='Top Genre', bbox_to_anchor=(1.05, 1), loc='upper left', fontsize=14, title_fontsize=12)

# Rotate x-axis labels for better readability
plt.xticks(rotation=45)
plt.grid(True)
plt.tight_layout()

# Show the plot
plt.show()

Section 3: Storytelling/Observations¶

Some Key Observations I have made by creating these visualizations are:

  • Younger users show a wider disparity in genre preferences with their primary preferenes towards Pop and EDM, with certain genres being more popular than others. As users age, the preferenes tend to level out with fewer extreme differences between genres

  • Music preferences evolve with age, with younger users favoring more energetic and mainstream generes, while older users lean toward more diverse range of genres, which tend to be more mellow. This suggests that age plays a role in shaping music preferences.

  • Genre preferences are influenced by cultural and regional factors. While some genres like Pop and EDM have global appeal, others are more localized. This shows the importance of considering cultural context when analyzing music trends and more than likely affects how much of that genre is created within that region as they know the people of that region prefer that genre more.

  • Time of day does influence genre preference, with certain genres aligning with specific activities or moods. For example, EDM and Rock are more popular with active times of the day while calming genres tend to dominate the times when people are winding down such as in the afternoon or nights (if they are not being active)

Section 4: Impact¶

  • Age-Based Recommendations: Streaming platforms can use these insights in orter to help users find content that matches their age group's music preferences and improve their overall recommendation algorithms based on factors like the user's age. where they live and the times in which they are listening to music and the genres in which they are listening to at these times to determine if they are a more active individual at specific times or not and use that to recommend content that is more appropriate for them on a personal level.

  • Time-Based Playlists: Create time-specific playlists to align with users preferences and create a more personalized experience for them.

  • Regional-Based Recommendations: Streaming platforms can curate region-specific playlists to cater the the cultural preferences.

  • Age-Specific Ad Campaigns- Music plaforms, artists and labels can promote concerts, merchandise, or ads for more energetic genres to appeal to younger audiences and more mellow genres to older audiences.

  • Genre-Specific Events: Organize events or festivals that focus on specific genres in specific regions where that genre is more popular in order to attract more people to those events.

  • Music Production and Artist Development Artists and record labels can use these insights to produce music that aligns with their audience's preferences region and focus on developing artists and labels that align with these preferences in order to attract more listeners and in turn increase their total revenue.

Section 5: Summary¶

This project explored how music preferences vary by age, country of origin, and time of day using a dataset of 5,000 global music streaming users. The analysis revealed that younger users gravitate toward energetic genres like Pop and EDM, while older users prefer more diverse and mellow genres such as Rock and Jazz. Regional differences show the cultural influence on music preferences, with certain genres dominating over others in specific countries. Additionally, time of day significantly impacts listening preferences such as genres, with energetic genres like EDM and Rock being popular during active periods (morning and nights some people are more active) and calming genres like Reggae dominating at night for those that are not active and are winding down. These insights have practical applications in personalizing music recommendations, optimizing marketing strategies, and planning events, ultimately enhancing user engagement and satisfaction in the music streaming industry.
